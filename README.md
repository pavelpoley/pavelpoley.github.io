## Fast PDF Reader

**Fast PDF Reader** is a lightweight Android app for viewing and managing PDF files in a simple and intuitive way. It focuses on fast performance, a clean interface, and practical tools for everyday reading and work with documents.

### Features

* View all PDF files in a simple and intuitive interface
* Search within PDFs to quickly find specific text
* Create and manage PDF bookmarks
* Save and edit quotes from PDFs
* Open password-protected PDFs
* Night and Sepia modes to reduce eye strain during reading
* Select and copy text from PDFs
* Switch between list and grid views with configurable columns
* Create shortcuts to frequently used PDFs on the home screen
* Browse PDFs organized by folders on the device
* Automatically remember the last opened page for each PDF

Simple PDF Reader is suitable for students, professionals, and anyone who regularly works with PDF documents and needs a stable, straightforward reader without unnecessary complexity.
